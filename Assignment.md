# Assignment: Notes Management App

In this assignment, students will develop a simple Notes Management Application using Flutter and Cloud Firestore.

The application should allow users to create, view, update, and delete notes. This project is designed to help students understand how Flutter applications interact with Firebase Firestore and how CRUD operations are performed in a real application.

## Project Requirements

Each note should contain:

- Title
- Description

Students must implement the following features:

### Create Note

Users can add a new note by providing a title and description.

### View Notes

All saved notes should be displayed in a list.

### Update Note

Users should be able to edit existing notes.

### Delete Note

Users should be able to remove notes from the database.

## Suggested Screens

### 1. Notes List Screen

Displays all notes stored in Firestore.

### 2. Add/Edit Note Screen

Allows users to create a new note or modify an existing one.
